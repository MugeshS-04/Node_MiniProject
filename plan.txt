User Table Model 
    - User id(PK) -> Integer
    - First Name -> String
    - Last Name -> String
    - Age -> Integer
    - Email -> String
    - Password -> String
    - Contact number -> String

Order Table Model
    - Order id(PK) -> Integer
    - User id(FK) -> Integer
    - Order Name -> String
    - Order Date -> Integer
    - Amount -> Integer
    - Payment Method -> String [Card / Cash]

Product Table Model 
    - Product id(PK) -> Integer 
    - Product name -> String
    - Product price -> Integer

Product_Order_Junction Table Model
    - id -> Integer
    - Product id(FK) -> Integer
    - Order id(FK) -> Integer    
    - Quantity -> Integer

User -> (One to Many) -> Orders
Orders -> (Many to Many) -> Products
